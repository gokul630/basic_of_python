List is a built-in data structure used to store an ordered collection of items. They are dynamic, resizable and capable of storing multiple data types.

Mutable: list elements can be changed, updated, added, or removed after the list is created.
Ordered: elements maintain the order in which they are inserted.
Index-based: elements are accessed using their position, starting from index 0.

Appends all items from an external collection to the end.insert(i, x): Inserts item x precisely at specified index position i.remove(x): Deletes the very first occurrence of value x.pop(i): Removes and returns the item at index i (defaults to the last item)
.clear(): Flushes the entire list, leaving it completely empty.index(x): Returns the integer index of the first matched value x.count(x): Counts the exact number of times value x appears.sort(): Reorders the internal components in ascending sequence natively.reverse(): Flips the current positional layout of all items in-place.copy(): Generates an independent shallow copy of the existing structure.
